Paste product slider photos in this folder.

Use these exact names:

- `product-1.webp`
- `product-1-details.webp`
- `product-2.webp`
- `product-2-details.webp`
- `product-3.webp`
- `product-3-details.webp`
- `product-4.webp`
- `product-4-details.webp`

Name rule:

- `product-N.webp` is the product photo shown in the phone and background carousel.
- `product-N-details.webp` is the matching mobile details screenshot shown below it.
- Keep the same number for matching pairs.

Example:

- San Pellegrino product photo: `product-1.webp`
- San Pellegrino detail UI screenshot: `product-1-details.webp`
